## How to 영어 기술면접 준비

### 1. Roles
`interviewee`
- 최소 2주에 한 번
- 한 주제에 대한 질문-답변을 준비하여 사전공유
- 받은 질문에 대해 script 없이 영어로 대답!

`interviewer`
- interviewee가 준비한 질문 중 랜덤으로 묻기
- 해당 분야 관련하여 다른 좋은 질문이 있다면 추가질문 가능!

### 2. Responsibilities
- interviewee에 대한 피드백 한 가지 이상

### 3. Schedule
- 매주 수, 10PM 구글밋
- ~2021.06.30까지 진행
